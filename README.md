This is the final project for Object Oriented Programming which takes the filepath of an image from the user and modifies it according to the user's prefrence 
with a list of filters given. Moreover, this project also has a user managing system where there are users and admins the users are required to login using their 
cnic and password which is cross checked with a file which stores their cnic and their password while registering the program also checks if the cnic registring is 
already registered or is blocked. The user after logging in applies filters from a list of 10 filters from an interactive menu in the console and then the image is saved back 
in the same folder from where it was recived
